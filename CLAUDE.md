# 维修派单管家

## 项目概述

维修派单管家是一个面向中小维修公司的全流程管理系统，解决"微信群里接单 → 电话派工 → 完工对不上账"的混乱现状。系统覆盖报修录入、智能派单、进度追踪、完工结算全流程闭环，让派单不漏单、月底秒对账、老板看数据。

## PRD 摘要

### 核心功能
- 工单管理：创建/派单/状态流转（待派单→进行中→已完工→已结算）
- 登录鉴权 + 角色权限：管理员全量操作，技师只看自己工单
- 数据看板：ECharts 折线图（近7天工单趋势）+ 饼图（工单状态分布）
- 技师管理：技师信息维护，派单时下拉选择
- 完工结算：录入维修费用，标记已结款
- 数据导出 Excel：导出工单列表为 .xlsx 文件
- 工单详情/备注：查看工单完整信息，追加维修备注
- 客户信息管理：常用报修客户快捷选择

### 目标用户
- **王姐**：派单员，每天接20-30个报修电话，需防漏单/重复派单
- **张师傅**：维修技师，需查看自己工单和结算金额
- **刘总**：公司老板，需看数据做决策、月底对账

### 技术栈
- 前端：Vue 3 + Element Plus + ECharts + Pinia + Vue Router
- 后端：Node.js + Express + better-sqlite3
- 认证：JWT (jsonwebtoken + bcryptjs)
- 数据库：SQLite
- 部署：pm2 + Nginx
